Serverless reduced operational burden, but it never fully removed infrastructure. This meetup talk explores superpositioned infrastructure and how Module Federation turns distributed JavaScript into a runtime-defined system.

The central idea is to decouple compute and code so services can compose dynamically instead of being tied to one fixed deployment surface.

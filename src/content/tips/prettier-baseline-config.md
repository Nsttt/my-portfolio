## The Config

Create `.prettierrc` in your project root:

```json
{
  "arrowParens": "always",
  "printWidth": 80,
  "singleQuote": false,
  "jsxSingleQuote": false,
  "semi": true,
  "trailingComma": "all",
  "tabWidth": 2
}
```

# Browse Néstor López Content

Use this skill when an agent needs to discover, cite, or navigate Néstor López's public technical writing, talks, projects, and tips on nstlopez.com.

## Resources

- Start with `https://nstlopez.com/llms.txt` for the compact content index.
- Use `https://nstlopez.com/llms-full.txt` when you need the full markdown export.
- Use `https://nstlopez.com/.well-known/openapi.json` for machine-readable endpoint discovery.
- Use `https://nstlopez.com/rss.xml` for chronological updates.

## Guidance

- Prefer the markdown representation by sending `Accept: text/markdown`.
- Cite canonical page URLs from `https://nstlopez.com/`.
- Treat content as public reference material. Respect the site's `Content-Signal` policy: `ai-train=no, search=yes, ai-input=yes`.
